La Inteligencia Artificial al Servicio de tus Códigos QR.

Lumi no es solo una aplicación de códigos QR; es una herramienta inteligente que democratiza la creación y el escaneo. Aprovechamos la potencia de la API de Gemini para transformar tus ideas en códigos QR visualmente impactantes y funcionalmente robustos, todo desde la palma de tu mano.

Escáner de Alta Precisión

Experimenta un escaneo ultrarrápido y fiable. Gracias a la integración con ML Kit de Google, Lumi decodifica códigos QR al instante.

Generador Impulsado por IA

Simplemente describe el código QR que deseas (contenido, colores, estilo) y nuestra integración con la API de Gemini lo hará realidad.

Integración con API de Gemini

El corazón inteligente de Lumi. La API de Gemini interpreta tus indicaciones de lenguaje natural, ofreciendo resultados sorprendentes.

UI Intuitiva y Responsiva

Diseñada con Jetpack Compose, Lumi ofrece una experiencia de usuario fluida y visualmente atractiva en cualquier dispositivo Android.
